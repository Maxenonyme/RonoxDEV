### 🎮 Game Development & Modding
Developing custom mechanics, rendering pipelines, and server-side optimization engines.

* 🛡️ **[Heracles](https://github.com/Maxenonyme/Heracles):** An aggressive server-side performance engine for NeoForge 1.21.1. It features AI task throttling, multi-threading via Java 21 Virtual Threads, dynamic block entity stasis, and client-side Dead Reckoning (Motion Extrapolation) to drastically reduce MSPT and network lag.
* 📹 **[Fathomcraft](https://github.com/Maxenonyme/Fathomcraft):** An immersive/horror mod featuring custom OpenGL shaders, VHS post-processing, bodycam HUD overlays, and dynamic story trigger blocks for advanced map making.
* 🛒 **[AdminShopSYS](https://github.com/Maxenonyme/AdminShopSYS):** A comprehensive, GUI-driven economy and shop management system allowing server administrators to edit prices and manage trades dynamically in-game.
